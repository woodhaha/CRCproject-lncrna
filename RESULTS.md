## 4. Elastic Net Feature Selection

```
Optimal parameters: α = 0.200  λ = 0.230  Min deviance = 2.588
```

**Interpretation**: The low α (0.200, closer to ridge than lasso) indicates that lncRNA expression features are **highly correlated** — biologically expected for co-expressed lncRNA networks sharing regulatory mechanisms. This near-ridge penalty retains correlated features rather than arbitrarily selecting one from each correlated group.

**17 lncRNAs** were selected with non-zero coefficients. After stepwise Cox refinement, **11 features** (8 lncRNAs + 3 stage dummy variables) achieved minimal AIC = 615.76 and C-index = 0.82 in training.
